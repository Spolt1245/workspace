Hardware Page

Motherboard

A motherboard is a board with circuits, that connects the
basic part of a computer so that it can function. It is where the
CPU, BUS, Disk drive, and the RAM connect. It is the
main part of the computer so it can work. The motherboard is a
medium size board that is usually green, but it depends. It has
small, and big intricate designs that connects all the parts of the
computer together.

History

  • Over many decades the motherboard has changed in size.
    During the time of its creation is was very big, and circuit printing
    wasn't very cheap. Now the motherboards have dropped in size, they are better made.

CPU (Central Proccessing Unit)

The CPU stands for central processing unit, which is the most
important part of the computer. It is the head of the computer.
This part of the computer makes all of the calculations required
for the computer. The CPU has small little chips that are built into it,
this allows the CPU to function. The CPU is not big in size, it is a
square chip that connects to the motherboard. There are tiny
connections that power it.

Current version

  • There are many times of CPU processors, but one that is the latest
    is Intel Xeon E5-2670, which is top of the line, because it is a
    very fast CPU. This CPU is cheap, but extremely effective to complete task fast.
